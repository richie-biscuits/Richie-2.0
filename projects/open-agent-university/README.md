# Open Agent University — Protocol Marketplace Concept

## What This Appears To Be
A web-based "university" interface where:
- **Professors** = Expert personas (Hormozi, Vaynerchuk, Tim Cook, etc.)
- **Protocols** = Extracted expertise as installable agent capabilities
- **The Experience** = Browse professors → View their protocols → "Enroll" (install into your agent)

## Visual Design Notes
- Dark mode, space grotesk font
- Card-based professor profiles with images
- Protocol cards with difficulty levels, duration, pricing
- Status dots showing "Active" state
- Hover effects and gradients
